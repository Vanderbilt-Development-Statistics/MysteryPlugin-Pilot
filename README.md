# MysteryPlugin-Pilot

## Step 1: Download IntelliJ IDEA
- Navigate to the JetBrains official website to download IntelliJ IDEA by going to this link: [IntelliJ IDEA Download](https://www.jetbrains.com/idea/download/other.html).
- Look for the "Community Edition" section. This version is free to use.
![IntelliJ Download](./Images/IntelliJDownload.png)
- Click on the download link for your operating system (Windows, macOS, or Linux).
- After the download is complete, run the installer and follow the on-screen instructions to install IntelliJ IDEA on your computer.
- Open IntelliJ IDEA. Upon launching, you might see a welcome screen if no projects are open.
![Welcome Page](./Images/WelcomePage.png)

## Step 2: Add the Plugin Repository
- Look for the "Plugins" button on the left side of the welcome screen and click it. If you already have a project open, navigate to **File** > **Settings** (on Windows/Linux) or **IntelliJ IDEA** > **Settings** (on macOS), and then select **Plugins** from the sidebar.
- Click on the **settings cog icon** located near the top of the window, beside the "Marketplace" and "Installed" tabs.
- From the dropdown menu, select **Manage Plugin Repositories...**
- Click **+** and add the following URL:
  ```
  https://vanderbilt-development-statistics.github.io/MysteryPlugin-Pilot/updatePlugins.xml
  ```
- Click **OK** to save.

## Step 3: Install the Plugin
- Go to the **Marketplace** tab in the Plugins window.
- Search for **"Development Statistics"**.
- Click **Install**.
![Plugin Installation](./Images/PluginInstallation.png)
- IntelliJ IDEA may require a restart to activate the plugin. If prompted, restart the IDE.

## Step 4: Enable the Plugin
- After installation, if the plugin is not automatically enabled, go to the **Installed** tab in the Plugins section.
- Find "Development Statistics" in the list. If it's disabled, click **Enable**.
![Enable Plugin](./Images/EnablePlugin.png)
- Restart the IDE if prompted.

## Step 5: Download and Open the Project Files
- Click the green **Code** button on this page and select **Download ZIP**.
![Repo Download](./Images/RepoDownload.png)
- Unzip the downloaded file and open the project folder in IntelliJ IDEA:
  - From the welcome screen, click **Open** and navigate to the project folder.
  - Or from an open project: **File** > **Open** and select the project folder.
